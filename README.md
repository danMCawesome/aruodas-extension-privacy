# Privacy Policy — Aruodas Assistant (Chrome Extension)

**Last updated:** 2025-07-12  
*(Replace this date with the current one)*

Aruodas Assistant (“the Extension”) is a Chrome browser extension designed to help users save notes, contact information, and reminders related to real estate listings on aruodas.lt.  
This Privacy Policy explains what information the Extension handles and how it is used.

---

## 1. Information the Extension Collects

The Extension does **not** automatically collect personal data.

However, users may manually enter the following information:

- Name and last name  
- Phone number (real or advertised)  
- Email address  
- Notes or comments  
- Callback dates and meeting information  

All such information is entered voluntarily.

The Extension does **not** collect any other personal or sensitive information.

---

## 2. How Information Is Used

All data entered into the Extension is used solely to provide core functionality:

- Saving notes for a listing  
- Displaying saved information when revisiting a listing  
- Showing callback reminders on the aruodas.lt homepage  

The Extension does **not** use data for:

- Advertising  
- Analytics  
- User tracking  
- Profiling  
- Sharing with third parties  

---

## 3. Where Data Is Stored

All user-provided data is stored **locally** on the user’s device using:

- `chrome.storage.sync` (synchronizes across the user's signed-in devices)
- `chrome.storage.local` (fallback)

The Extension does **not** transmit data to the developer or to external servers.

---

## 4. Data Sharing

The Extension:

- Does **not** share data with the developer  
- Does **not** send data to remote servers  
- Does **not** share data with advertisers, analytics providers, or external organizations  
- Does **not** sell personal data  

All data stays on the user’s device.

---

## 5. Website Interaction

The Extension interacts only with pages on **aruodas.lt**.

It reads limited page content (such as listing text or displayed phone number) only to pre-fill fields or display UI elements.  
No website data is stored or transmitted outside the user’s device.

---

## 6. User Control and Data Deletion

Users can delete stored data at any time by:

- Clicking **Remove** in the extension toolbar on a listing page  
- Clicking **Remove** in the “Today’s callbacks” section  
- Uninstalling the extension, which removes all locally stored data  

---

## 7. Children’s Privacy

This Extension is not intended for use by children under 13.  
It does not knowingly collect data from children.
